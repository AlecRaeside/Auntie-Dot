Auntie-Dot by AlecRaeside

loading animation from Halo: Reach recreated with html, css and js

The Xbox 360 game "Halo: Reach" developed by Bungie displays a grid of diagonal rectangles when something is loading. I have tried to simulate this using modern web technologies
